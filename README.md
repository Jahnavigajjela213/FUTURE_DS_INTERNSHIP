# Future Interns — Data Science & Analytics Internship

This repository contains the tasks I completed during my **Data Science & Analytics Internship** with [Future Interns](https://futureinterns.com).  
The internship focused on applying **data analysis, visualization, and NLP techniques** to real-world problems in business and education.


## 📌 Internship Details
- **Organization:** Future Interns  
- **Domain:** Data Science & Analytics (Track Code: DS)  
- **Format:** Self-paced, project-based internship  
- **Deliverables:** 3 tasks with GitHub submission & LinkedIn posts

## 📝 Tasks Overview

### 🔹 Task 1: Business Sales Dashboard
- **Objective:** Analyze e-commerce data to identify top products, sales trends, and high-revenue categories.  
- **Skills:** Data cleaning, visualization, storytelling.  
- **Key Insight:** Electronics (Phones & Laptops) generated the highest revenue.  
- **Notebook:** `Task1_Business_Sales_Dashboard.ipynb`  


### 🔹 Task 2: Social Media Campaign Performance Tracker
- **Objective:** Evaluate ad campaigns (Facebook & Instagram) using CTR, CPC, ROI & engagement metrics.  
- **Skills:** Marketing analytics, KPI calculation, visualization.  
- **Key Insight:** Instagram campaigns had higher CTR, but the Facebook Festival campaign achieved the best ROI.  
- **Notebook:** `Task2_Social_Media_Campaign_Tracker.ipynb`  


### 🔹 Task 3: College Event Feedback Analysis
- **Objective:** Perform sentiment analysis on student survey feedback to uncover satisfaction trends and improvement areas.  
- **Skills:** NLP (TextBlob), survey insights, visualization.  
- **Key Insight:** Positive feedback praised networking & workshops, while negative feedback highlighted food & crowding issues.  
- **Notebook:** `Task3_Event_Feedback_Analysis.ipynb`  


## 🚀 Tools & Technologies

- **Environment:** Google Colab / Jupyter Notebook  
- **Other Tools:** Power BI, Excel (optional for dashboards)
